# ⌨️ CEBT – CLI for End Users

One-command installers for Windows, macOS, and Linux.

---

## 📥 Installers

| OS | File | What to Do |
|----|------|------------|
| Windows | [`install-windows.bat`](./install-windows.bat) | Double-click to install |
| macOS | [`install-macos.sh`](./install-macos.sh) | Run in terminal: `bash install-macos.sh` |
| Linux | [`install-linux.sh`](./install-linux.sh) | Run in terminal: `bash install-linux.sh` |

---

## 🚀 After Installation

Open any terminal and type:

```bash
cebt /S     # Start test
cebt /R     # Show results
cebt /C     # Compare tests
cebt /L     # Open last report
cebt /O     # Set output folder
cebt /H     # Show help
